# Phase 3

## デザイン

- shadcnを利用する。
  - StyleはLyra
  - ThemeはBlue
  - Fontは英文はRoboto, 和文はNoto Sans
  - IconはLucide
- スマートフォンで使いやすくする。PCで見ることはない。
- UIに日本語は使わない。英語を使う。

## 仕様

- 「Let's get started!」と表示。startボタンがある。startボタンをタップすると次のページを表示。
- ページを開くとAPIから1件レコードを取得して表示する
- 英単語とAnswerボタンを表示する
- Answerボタンをタップすると日本語訳、品詞、例文が表示される。例文訳は不要です。
- Nextボタンをタップすると次のレコードを取得して表示。
- これを繰り返す。
- カウントを表示する。
- 前回と同じ単語レコードの場合はもう一度APIをフェッチする
- 解答ページ Finishボタンをつけて、タップすると今日の結果ページへ進む。
- 結果ページは、今日やった単語の一覧が並ぶ。あと何個やったか。
- リロードすると最初に戻る。

## 技術仕様

- マジックナンバーは定数にする
- APIエンドポイントは定数にする
- 余計なライブラリは利用しない。ミニマムで。
  - ただし時間操作が必要な場合はdate-fns, date-fns/localeを使う。
- React公式の推奨実装方法に従う
